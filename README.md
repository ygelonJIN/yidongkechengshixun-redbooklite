# Dice Roller

一个使用 Jetpack Compose 构建的交互式骰子掷骰 Android 应用。

## 功能说明

- 点击 **Roll** 按钮随机掷出骰子（1~6 点）
- 骰子图片实时更新，显示对应点数的骰面
- 使用 
emember 和 mutableStateOf 管理状态，支持 Compose 重组

## 技术栈

- **语言**: Kotlin
- **UI 框架**: Jetpack Compose (Material 3)
- **最低 SDK**: API 24 (Android 7.0 Nougat)
- **目标 SDK**: API 34

## 运行方式

1. 用 **Android Studio** 打开本项目根目录（DiceRoller/）
2. 等待 Gradle 同步完成
3. 连接 Android 设备或启动模拟器（API 24+）
4. 点击 Run 运行应用
